# Método Expansão de Cofatores
- O método expansão por cofatores ou "desenvolvimento de Laplace" é um método que permite calcular o determinante de uma matriz quadrada de qualquer ordem.  
$$
\det(A) = \sum_{j=1}^{n} a_{ij} C_{ij} \quad \text{(expansão pela i-ésima linha)}
$$
  
$$
\det(A) = \sum_{i=1}^{n} a_{ij} C_{ij} \quad \text{(expansão pela j-ésima coluna)}
$$
